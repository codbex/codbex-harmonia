# Spinner

The spinner component informs the user about an ongoing operation. Similar to progress but it does not show how much is done or is left to be done.

## API Reference

### Component attubute(s)

```
x-h-spinner
```

## Examples

<ClientOnly>
<component-container data-class="flex flex-col items-center gap-4">
<span x-h-spinner></span>
<span x-h-spinner class="size-8"></span>
<span x-h-spinner class="size-12"></span>
</component-container>
