# Rotate

CSS utility classes to rotate elements.

## Class names

| Class      | Description         |
| ---------- | ------------------- |
| rotate-90  | Rotate 90 degrees.  |
| rotate-180 | Rotate 180 degrees. |
| rotate-270 | Rotate 270 degrees. |
